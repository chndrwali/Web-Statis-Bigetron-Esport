Sejarah

Bigetron Esport resmi memperkenalkan Red Aliens pada tanggal 28 Mei 2018 dengan merekrut Zuxxy,Luxxy,Natic dan Kingz. Nama Bigetron langsung melambung usai Menjuarai PINC 2018 dan menjadi Wakil Indonesia di kejuaraan SEA di event PUBG Mobile Start Chalange 2018.Tidak lama setelah itu Bigetron di perkuat Ryzen

Dominasi Bigetron mendominasi di tahun 2019 setelah menjuarai Turnamen Official Tencent di event PMCO Spring Split 2019,namun gagal menjuarai kembali PINC 2019 setelah mereka berhasil finish di peringkat ke dua,setelah itu kembali Bigetron kehilangan Roster nya yaitu sang kapten Natic. namun sebgai ganti nya MICROBOY kini memperkuat Bigetron lalu berubah nama lah mereka menjadi Bigetron Red Aliens. tidak cukup sampai disitu Bigetron merekrut Boyka menjadi pelatih yang di kenal sebagai pemain AROV tersbut kini resmi berjuang dengan Pasukan Red Aliens setelah terseok di turnamen SEA kini bigetron menggapai sebuah mimpi mereka setelah berhasil menjuarai PMCO Spring Split 2019 Global Finals di Malaysia
